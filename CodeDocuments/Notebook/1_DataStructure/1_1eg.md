## 实例1

```cpp
struct
{
    int no ;
    char name[ 8 ] ;
    char sex[ 2 ] ;
    char class[ 4 ] ;
} Stud[ 7 ] = { { 1 , "张斌" , "男" , "9901" } , ... , { 5 , "王萍" , "女" , "9901"}} ;
```

p.s 其中  
Stud[7]就相当于常规意义上的数组，数组名和数组下标的用法无差别

## 实例2

常见的单链表结构

```cpp
typedef struct Studnode
{ 
    int no = 0 ;
    char name [8] ;
    char sex[2];
    char class [4] ;
    struct Studnode* next = NULL ;//指向下一个学生结点的指针
} StudType;//结点类型，即结点的标识名称
```

一般，首结点以head命名，尾结点应该指向NULL(空)
